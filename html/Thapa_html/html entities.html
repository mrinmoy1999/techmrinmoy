HTML ENTITIES

html entities are code used to represent special characters and
symbols that have reserved meaning in HTML.These entities are especially
importants when you want to display characters that might conflict with
HTML syntex or when you want to display characters that aren't directly
available on your keyboard.HTML entities are represented using an ampersand
(&)followed by a code and a semicolon(;).
